---
title: Data types
description: Data types supported by oRPC for seamless transformation between client and server.
---

- `string`
- `number` (include `NaN`)
- `boolean`
- `null`
- `undefined`
- `Date` (include `Invalid Date`)
- `BigInt`
- `RegExp`
- `URL`
- `Set`
- `Map`
- `Blob`
- `File`

> **Note**: Only apply when transferring data between client and server.
