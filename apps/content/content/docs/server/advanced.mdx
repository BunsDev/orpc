## Internal Caller

```ts
import { initORPC, createRouterCaller } from '@orpc/server'

const orpc = initORPC

const appRouter = orpc.router({
    ping: orpc.handler(() => 'pong')
})

const caller = createRouterCaller({
    router: appRouter,
    context: {}
    validate: true // set false if you want bypass the validation input and output
})

const result = await caller.ping() // result is 'pong'
```

## Transformer

oRPC supports [SuperJSON](https://github.com/flightcontrolhq/superjson) as a transformer out of the box.
If you want to use other transformers, you can use the `transformer` option.

```ts
const newTransformer = {
    stringify: (data) => SuperJSON.stringify(data),
    parse: (data) => SuperJSON.parse(data),
}

// on server
fetchHandler({
    request,
    handler,
    prefix: '/',
    context: {},
    transformer: newTransformer,
})

// on client
const client = createORPCClient({
  baseURL: 'http://localhost:2026',
  transformer: newTransformer
})
```

The transformer in client must be the same as the one in server.
